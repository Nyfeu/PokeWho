# Quickstart

## 1. Criar o ambiente virtual

```bash
python3 -m venv .venv
```

## 2. Ativar o ambiente virtual

### Linux/macOS
```bash
source .venv/bin/activate
```

### Windows
```bash
.venv\Scripts\activate
```

## 3. Instalar dependências

```bash
pip install -r requirements.txt
```

## 4. Executar a aplicação

```bash
python main.py
```
